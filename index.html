    Google 試算表同步設定

    📝 如何取得網址？(只需設定一次)

    1. 新增一個空白的 Google 試算表。
    2. 點擊上方選單的 「擴充功能」 > 「Apps Script」。
    3. 將編輯器內的程式碼清空，貼上以下程式碼： 
      function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);

  sheet.clear();
  sheet.appendRow(['任務ID', '內容', '預定日期', '狀態']);

  data.tasks.forEach(function(t) {
    sheet.appendRow([t.id, t.text, t.date || '今日', t.completed ? '已完成' : '未完成']);
  });

  return ContentService.createTextOutput("Success");
}
    4. 點擊右上角 「部署」 > 「新增部署作業」。
    5. 類型選擇 「網頁應用程式」。
    6. 存取權限請選擇 「所有人」，然後點擊「部署」。
    7. 授權完成後，複製產生的 「網頁應用程式網址」，貼到上方的輸入框即可！
